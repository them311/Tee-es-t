LFDS Quiz — Intégration sur l-fds.com

Trois façons d'ajouter le quiz « Quel gourmet êtes-vous ? » sur votre site La Française Des Sauces.

URL du quiz : remplacez https://lfds-quiz.netlify.app par l'URL réelle de votre site Netlify (ou par https://quiz.l-fds.com si vous configurez un sous-domaine).

1 — Bouton CTA (recommandé)

À coller n'importe où dans votre page (header, hero, footer…). Le visiteur arrive sur le quiz dans un nouvel onglet.

3 — Embed iframe (quiz directement dans la page)

Pour intégrer le quiz à l'intérieur d'une page existante de l-fds.com, sans redirection.

<iframe src="https://lfds-quiz.netlify.app/quiz/"
        title="Quiz LFDS — Quel gourmet êtes-vous ?"
        style="width:100%;height:780px;border:none;
               border-radius:12px;
               box-shadow:0 8px 32px rgba(0,0,0,0.08);"
        loading="lazy">
</iframe>

4 — Tracking UTM (optionnel mais recommandé)

Ajoutez des paramètres UTM au lien pour identifier d'où viennent vos visiteurs dans le dashboard admin.

https://lfds-quiz.netlify.app/quiz/?utm_source=l-fds.com&utm_medium=hero&utm_campaign=quiz_2026

Ces UTM sont automatiquement capturés par le quiz et apparaissent dans /api/admin/submissions.

5 — Sous-domaine personnalisé (recommandé)

Pour une URL plus propre comme quiz.l-fds.com au lieu de lfds-quiz.netlify.app :

  1. Dans Netlify : Site settings → Domain management → Add custom domain → entrez quiz.l-fds.com
  2. Dans votre DNS (chez votre registrar du domaine l-fds.com), ajoutez un enregistrement : 
    Type: CNAME
Nom:  quiz
Cible: lfds-quiz.netlify.app
  3. Netlify provisionne automatiquement le certificat HTTPS via Let's Encrypt (≈ 1 minute).
  4. Remplacez ensuite tous les liens ci-dessus par https://quiz.l-fds.com/quiz/.
