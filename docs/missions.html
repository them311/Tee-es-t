⚙ Comment generer une reponse
Workflow GitHub Actions

Pour generer une reponse adaptee a une mission, lancez le workflow GitHub Generer reponse de mission :

  1. Copiez le texte complet de la mission depuis Codeur, Malt, Upwork, etc.
  2. Ouvrez le workflow sur GitHub et cliquez sur Run workflow.
  3. Selectionnez la plateforme cible (codeur, malt, upwork, fiverr, linkedin, freelance-info, other).
  4. Collez le texte de la mission dans le champ mission_text.
  5. Optionnel : indiquez un budget et un niveau d'urgence.
  6. Lancez. Apres 20-40 secondes, la reponse apparait dans la liste ci-dessous.

La reponse respecte le ton, la longueur et la structure attendus sur la plateforme cible. Elle contient un devis chiffre en phases et se signe Baptiste Thevenot. Aucun caractere special ni emoji.
