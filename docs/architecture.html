📜 Composants
6 services connectes
C
Cerveau

Claude AI (Anthropic)

Modele de raisonnement qui orchestre chaque routine : analyse du contexte, redaction des emails, qualification des deals, prise de decision.

H
CRM

HubSpot

Source de verite pour les contacts, companies et deals. L'agent y lit les fiches et y pousse les mises a jour de stage, les notes et les taches.

G
Email

Gmail API

Lecture des reponses entrantes et creation de brouillons. L'agent ne fait jamais d'envoi silencieux — tout passe par des drafts valides.

N
Documentation

Notion

Workspace de travail pour les comptes rendus de reunion, les audits hebdomadaires et la base de connaissance produit.

❯
Orchestration

GitHub Actions

Runner gratuit qui declenche les 3 routines selon un cron. Versionne tout le code agent, les logs et les rapports.

N
Hebergement

Netlify

Sert ce dashboard public en statique. CDN global, CSP strict, MAJ automatique a chaque push.

🔒 Securite & donnees

Credentiels

Toutes les cles API (HubSpot, Gmail, Notion, Anthropic) sont stockees en tant que GitHub Secrets chiffres, jamais ecrites dans le repo.

Perimetre d'ecriture

L'agent ne peut ecrire que dans les objets dont il est owner. Les envois emails passent par des drafts valides manuellement.

Tracabilite

Chaque action est loggee dans la timeline HubSpot et versionnee dans Git. L'historique est public.
