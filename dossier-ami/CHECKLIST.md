# Checklist — pièces à rassembler avant d'envoyer

Coche au fur et à mesure. Range chaque pièce dans le bon sous-dossier de
`pieces/`. Numérote les fichiers (`01-cni.pdf`, `02-justif-domicile.pdf`...)
pour qu'ils ressortent dans le bon ordre une fois imprimés.

## Identité (`pieces/identite/`)

- [ ] Pièce d'identité recto-verso (CNI ou passeport)
- [ ] Justificatif de domicile de moins de 3 mois (facture énergie / télécom / quittance de loyer)
- [ ] RIB
- [ ] (Si tutelle / curatelle) Jugement et pièce d'identité du représentant

## Contrats et conventions (`pieces/contrats/`)

- [ ] Contrat Cetelem signé (offre de prêt + conditions générales)
- [ ] Convention de compte bancaire
- [ ] Contrat opérateur (mobile / internet)
- [ ] Contrat d'assurance (CG + CP)
- [ ] Tout avenant signé

## Relevés et facturation (`pieces/releves/`)

- [ ] 12 derniers relevés bancaires (PDF officiels)
- [ ] Échéancier Cetelem à jour
- [ ] 6 dernières factures opérateur
- [ ] Quittances d'assurance des 24 derniers mois
- [ ] Relevés FICP / FCC Banque de France (à demander gratuitement à la BDF)

## Correspondances antérieures (`pieces/correspondances/`)

- [ ] Tout email envoyé / reçu (export PDF avec en-têtes)
- [ ] Tout courrier postal scanné recto-verso
- [ ] Captures des messageries sécurisées (banque, opérateur)
- [ ] Captures de SMS pertinents (avec date / numéro visible)
- [ ] Notes d'appel téléphonique (date, heure, n° appelé, durée, interlocuteur, résumé)

## Preuves matérielles (`pieces/preuves/`)

- [ ] Captures d'écran horodatées (espace client, app)
- [ ] Photos (matériel, document, lieu)
- [ ] Attestations de tiers (datées, signées, avec copie CNI du témoin)
- [ ] PV / dépôt de plainte / main courante (si applicable)
- [ ] Constats d'huissier (si applicable)

## Avant d'envoyer — relecture finale

- [ ] Toutes les `{VARIABLES}` des templates sont remplies
- [ ] Les pièces jointes citées dans le courrier existent **et** sont jointes
- [ ] Le courrier est daté, signé, paginé
- [ ] L'adresse du destinataire est exacte (vérifier sur le site officiel)
- [ ] La copie est conservée dans `archives/`
- [ ] L'envoi est loggé dans `SUIVI.md`
